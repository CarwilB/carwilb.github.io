Cleaning up bare URLs

The Problem

In researching the Ultimate Consequences database, we’ve introduced numerous URLs as references for particular entriess without including them in our Zotero library.

In Narrative-Status.qmd, we search for all URLs in our datasheet and then create a list of those that are not in the Zotera library, remaining_unreferenced_urls.

The goal is to find citable webpage forms of these URLs, either still online or on Archive.org’s Wayback Machine

Importing and cleaning URLs

The source list (urls-unreferenced-2026-03-06.txt) contains bare URLs extracted from the narrative datasheet that have not yet been added to the Zotero library. We strip trailing punctuation, split any lines where two URLs were accidentally concatenated, and deduplicate.

Code 
raw_lines <- readLines("../../ultimate-consequences/data/urls-unreferenced-2026-03-06.txt")

urls_fixed <- raw_lines |>
  trimws() |>
  sub("[.;,\"!]+$", "", x = _) |>
  trimws() |>
  keep(\(u) nchar(u) > 0) |>
  map(\(u) str_extract_all(u, "https?://[^\\s]+")[[1]]) |>
  unlist() |>
  unique()

Scan results

We checked each URL against the live web and the Wayback Machine CDX API. Results were saved externally and are loaded here. Redirected URLs that share a single destination (e.g. a newspaper that migrated domains) are flagged as “merged” — they are treated separately since their original URLs are no longer canonical.

classify_url <- function(url) {
  for (nm in names(newspapers)) {
    if (str_detect(url, newspapers[[nm]])) return(nm)
  }
  if (str_detect(url, "indymedia"))    return("Indymedia")
  if (str_detect(url, "google\\.com")) return("Google")
  tld <- str_extract(url, "(?<=\\.)edu|(?<=\\.)org|(?<=\\.)gov")
  if (!is.na(tld)) return(paste0(".", tld))
  return("Other")
}
